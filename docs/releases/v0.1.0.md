# GitWhisper v0.1.0

GitWhisper v0.1.0 is the first public release of the Rust CLI for Git commit intelligence.

## Highlights

- Capture commit context from Git, commands, environment, IDE hints, and review signals.
- Explain why files changed using Git history plus Gemini, Ollama, hybrid model selection, or heuristic fallback.
- Analyze code health with quality, security, performance, bug prediction, knowledge risk, and refactor priority reports.
- Share knowledge through Git notes, Slack, Discord, GitHub PR summaries, GitLab MR summaries, wiki generation, and ADR generation.
- Store feedback and audit events locally with JSON or in PostgreSQL for team/Docker workflows.
- Run a local dashboard and export analytics snapshots as JSON or CSV.

## Installation

```bash
cargo install --path .
```

Or build from source:

```bash
cargo build --release
```

## First Run

```bash
gitwhisper init
gitwhisper capture
gitwhisper annotate
gitwhisper explain src/main.rs
```

## Docker

```bash
docker compose up --build
```

The dashboard runs on `http://localhost:7878`.

## Validation

- Unit tests: 26 passing.
- PostgreSQL backend: live-tested with Docker Compose.
- Feedback export: JSON and CSV tested.
- Audit prune/log: tested on JSON and PostgreSQL paths.

## Known Limitations

- Advanced analyzers are heuristic in this release.
- Full SSO, advanced RBAC, distributed workers, and hosted cloud deployment are future work.
- External publishing integrations require user-provided credentials and project-specific configuration.
